Abstracts

O trabalho que foi necessário efetuar para o quarto TPC de PRI relaciona-se com desenvolvimento de um servidor capaz de responder a métodos GET provenientes do lado do cliente para obter informação sobre um determinado documento XML referente a uma dada música. Partindo de um XML inicial, arq-son.xml, efetuou-se a sua divisão por tags doc, usando para isso a stylesheet arq.xsl. Esta divisão foi efetuada, tendo como base o trabalho desenvolvido na aula de dia 21/7/2019 pelo professor José Ramalho.

Relativamente ao servidor, trataram-se, num ficheiro .js, os diferentes tipos de pedidos que o este poderia receber. Assim, foram especificados os casos em que é pedido um determinado ficheiro XML, por exemplo, doc1.xml. Para se apresentar o ficheiro ao "cliente", é necessário saber o número que foi indicado no pedido GET. Ao texto indicado na barra de endereços, é aplicado um filtro( split), que retira do mesmo o identificador do ficheiro XML. Este ficheiro XML é lido, sendo posteriormente formado um pacote de dados com a informação lida. De seguida, foi necessário fazer a conversão do ficheiro XML para HTML. Existe uma stylesheet, designada por doc2html, que para nodos associados às tags XML de um dado documento, aplica instruções HTML aos mesmos. Sendo assim, também foi necessário especificar o caso de o pedido GET conter a descrição do doc2html. Por fim, definiu-se que o servidor também estaria apto para tratar de pedidos do ficheiro w3.css, nos casos em que é necessário aplicar certos formatos e templates ao texto HTML. Nestes dois últimos casos, foram igualmente formados pacotes de dados com a informação lida dos ficheiros indicados.
