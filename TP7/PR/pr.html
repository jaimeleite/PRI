Abstracts

O trabalho que foi necessário efetuar para o sexto TPC de PRI relaciona-se com o desenvolvimento em NodeJs, apoiado por ExpressJS, de um servidor capaz de responder a pedidos provenientes do lado do cliente para registar/consultar/eliminar filmes de uma base de dados em MongoDB.

Como resposta às especificações para este TPC, o servidor é capaz de: responder a pedidos para apresentar a página principal ( GET /), no roteador index; no roteador filmes são respondidos aos pedidos GET /filmes, GET /filmes/:id, GET /filmes/inserir, POST /filmes e DELETE /filmes/:id. Para estes pedidos, é apresentada a página com todos os filmes, é pesquisado um dado filme, é apresentado o formulário relativo ao registo de um dado filme, inserir um filme na base de dados e remover um dado filme da base de dados, respetivamente. Já no roteador api, são analisados os pedidos GET /api/filmes, GET /api/filmes/:id, POST /api/filmes, DELETE /api/filmes/:id. Este roteador faz a ligação com a base de dados e os pedidos indicados anteriormente relacionam-se com a visualização de todos os filmes, a visualização de um dado filme, inserir um dado filme na base de dados e apagar um determinado filme da base de dados, respetivamente.
