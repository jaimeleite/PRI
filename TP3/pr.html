Abstract

Document Type Definition, mais conhecido por DTD, é o que define um dado documento da família SGML(Standard Generalized Markup Language). Para este documento, é apresentada a definição da sua estrutura, indicando os elementos e atributos necessários para a sua construção. Neste caso, e seguindo a ideia proposta pelo professor José Ramalho, é necessário definir um documento XML(que repesenta a estrutura de um manifesto), usando para isso um DTD. Sendo assim, vista a informação que é útil retratar neste tipo de ficheiros(manifesto), a descrição dos vários TPC's a entregar terá a seguinte estrutura: informações gerais, com a identificação do número do TPC, bem como título, subtítulo(opcional) e intervalo de datas em que se encontra o TPC, entre outras; lista dos trabalhos a entregar, que será representada por um link para um repositório GitHub e para os trabalhos nesse mesmo repositório(referentes ao dado TPC); resumo do que o TPC em questão aborda; identificação dos alunos que desenvolveram o TPC. XPathé uma linguagem que permite selecionar partes(nós) de um dado documento XML. Na segunda parte do segundo TPC, foi proposto que, partindo de ficheiros no formato XML, se respondesse a várias queries.
