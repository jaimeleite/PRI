Abstracts

O trabalho que foi necessário efetuar para o terceiro TPC de PRI relaciona-se com a especificação de uma stylesheet para transformar o PR.xml em PR.html, bem como especificar uma stylesheet para gerar um website para os arqueossítios.

Na aula de dia 7 de outubro de 2019, o professor José Ramalho definiu, em conjunto com a turma, o DTD que serve de suporte para a definição do documento XML do manifesto. Até ao final do semestre, é obrigatório usar esse mesmo DTD para apresentar os vários TPC's.

Sendo assim, para gerar a página HTML do pr, apoiado pela definição do DTD, definiu-se um ficheiro .xsl no qual, para os vários nodos do XML do manifesto, aplicam-se normas HTML por forma a representar de forma correta e legível toda a informação.

Em relação ao website para os arqueossítios, foi importante, mais uma vez, o trabalho desenvolvido na aula de dia 7 de outubro de 2019 pelo professor José Ramalho, que efetuou a especificação de um ficheiro .xsl, criando uma página principal( index.html) e respetivas derivações("ficheiros auxiliares"). Assim para gerar um website para os arqueossítios, foi importante criar um index.html, no qual existem links que fazem a ligação com a descrição de cada arqueossítio referenciado nesses mesmos links. Tudo isto foi possível com a definição do ficheiro arq.xsl, que tem instruções HTML para cada nodo do ficheiro arq.xml.
