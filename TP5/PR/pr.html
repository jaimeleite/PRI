Abstracts

O trabalho que foi necessário efetuar para o quinto TPC de PRI relaciona-se com o desenvolvimento em NodeJs, apoiado por ExpressJS, de um servidor capaz de responder a pedidos provenientes do lado do cliente para registar/consultar/eliminar alunos de uma base de dados, que é persistida por um ficheiro em JSon.

Como resultado do trabalho desenvolvido, o servidor é capaz de responder a métodos GET, POST e DELETE. Relativamente ao primeiro método referenciado, é possível apresentar ao cliente todos os alunos presentes na base de dados ( GET /alunos), bem como a informação relativa a um determinado aluno ( GET /alunos/:idAluno). Relativamente ao método POST, é possível atualizar a base de dados com um novo aluno ( POST /alunos) ou ainda atualizar as notas de um determinado aluno( POST /alunos/:idAluno/notas). Relativamente ao método DELETE, o servidor é capaz de tratar da eliminação de um determinado aluno da base de dados( DELETE /alunos/:idAluno)) ou ainda de apagar uma nota de um aluno( DELETE /alunos/:idAluno/notas/:indicador).
