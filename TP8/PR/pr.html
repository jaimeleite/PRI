Abstracts

O trabalho que foi necessário efetuar para o sétimo TPC de PRI relaciona-se com o desenvolvimento em NodeJs, apoiado por ExpressJS, de um servidor capaz de responder a pedidos provenientes do lado do cliente para consultar obras de uma base de dados em MongoDB.

Como resposta às especificações para este TPC, o servidor é capaz de responder a vários pedidos de GET, tais como: consultar todas as obras da base de dados ( "GET /obras"); consultar as obras de um determinado periodo( "GET /obras?periodo=x"); consultar as obras com um dado ano de criação ( "GET /obras?anoCriacao=y")); consultar todas as obras que pertencem a um compositor( "GET /obras?compositor=A" / "GET /compositores/:nome"); consultar todos os compositores( "GET /compositores"). Este servidor apenas tem um roteador para responder aos pedidos.
