Abstracts

O trabalho que foi necessário efetuar para o oitavo TPC de PRI relaciona-se com o desenvolvimento em NodeJs, apoiado por ExpressJS, de um servidor capaz de responder a pedidos provenientes do lado do cliente para consultar ficheiros de uma base de dados em MongoDB. Os resultados destas consultas são expressos por modais, em que a informação sobre os ficheiros selecionados são apresentados nos mesmos(no caso de imagens, estas aparecem nos modais).

Os clientes deste servidor também conseguem fazer UPLOAD de ficheiros, sendo estes movidos de um determinado local do seu dispositivo para uma pasta específica onde está a correr o projeto. Além disso, os clientes conseguem visualizar todos os ficheiros contidos na base de dados.
